## 🔐 1. 로그인 화면

**URL**: `/login`

### 레이아웃
```
┌────────────────────────────────────┐
│                                    │
│         [42ment Logo]              │
│                                    │
│  ┌──────────────────────────────┐  │
│  │ Username:                    │  │
│  │ [__________________]         │  │
│  │                              │  │
│  │ Password:                    │  │
│  │ [__________________]         │  │
│  │                              │  │
│  │     [  로그인  ]              │  │
│  └──────────────────────────────┘  │
│                                    │
└────────────────────────────────────┘
```

### 필드
| 필드명 | 타입 | 검증 | 필수 |
|--------|------|------|------|
| Username | text | 5-20자 영문/숫자 | Y |
| Password | password | 8-30자 | Y |

### 동작
1. 입력값 검증 (클라이언트)
2. API 호출: `POST /auth/login`
3. 성공 시: JWT 토큰 저장 → 대시보드로 이동
4. 실패 시: 에러 메시지 표시

### 에러 메시지
- "아이디 또는 비밀번호를 확인해주세요"
- "네트워크 오류가 발생했습니다"

### 기능
1. **검색**: 회사명, 담당자명, 전화번호로 검색
2. **필터**: 상태별 (전체/활성/비활성/정지)
3. **페이지네이션**: 20개씩
4. **엑셀 다운로드**: CSV/Excel 선택
5. **행 클릭**: 고객 상세 페이지로 이동

### 컬럼
- 회사명
- 담당자
- 전화번호
- 상태 (뱃지: 활성=녹색, 비활성=회색, 정지=빨강)

### API 호출
- `GET /clients?page={page}&limit=20&search={search}&status={status}`

| 필드명 | 타입 | 검증 규칙 | 필수 |
|--------|------|-----------|------|

| 수신 이메일 | text | 이메일 형식 (쉼표 구분) | Y (활성화 시) |
| 템플릿 | select | 템플릿 목록 | Y (활성화 시) |

1. 입력값 검증
2. API 호출:
   - 등록: `POST /clients`
   - 수정: `PUT /clients/:id`
3. 성공 시: 고객 상세 페이지로 이동
4. 실패 시: 에러 메시지 표시
