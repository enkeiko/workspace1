# 광고대행사 관리 시스템 - 단계별 구축 로드맵

**프로젝트명**: 42ment 광고대행사 관리 시스템
**대상**: 1인 또는 소규모 광고대행사
**버전**: v1.0
**작성일**: 2025-11-14

---

## 📌 프로젝트 개요

### 목표
1인 또는 소규모 광고대행사가 고객, 매장, 판매, 구매, 정산을 체계적으로 관리하고, 업무 자동화를 통해 생산성을 향상시킬 수 있는 통합 관리 시스템을 구축합니다.

### 핵심 원칙
- **데이터 연동**: 모든 정보는 서로 연동되어 중복 입력 최소화
- **점진적 자동화**: API 연동이 어려운 부분은 수동 또는 엑셀 업로드로 시작
- **데이터 주권**: 모든 목록 및 데이터는 다운로드 가능
- **확장성**: 필요에 따라 기능 추가 가능한 모듈형 구조

---

## 🎯 Phase별 개요

| Phase | 기간 | 목표 | 핵심 가치 |
|-------|------|------|-----------|
| **Phase 1** | 3-4개월 | MVP - 핵심 비즈니스 운영 | 일상 업무를 시스템으로 전환 |
| **Phase 2** | 2-3개월 | 효율화 & 자동화 | 반복 작업 자동화 |
| **Phase 3** | 2-3개월 | 고급 기능 & 외부 연동 | 데이터 기반 의사결정 |

## 📊 Phase 1: MVP - 핵심 비즈니스 운영 (3-4개월)

### 목표
일상적인 광고대행사 업무를 시스템으로 관리할 수 있는 최소 기능 구현

### 개발 우선순위

#### 🔴 최우선 (1-2개월)

##### 1. 내정보 관리
- 회사 기본 정보 (상호, 대표자, 사업자번호, 연락처)
- 회사 계좌 정보
- 사업자등록증 파일 업로드

##### 2. 고객 관리 (기본)
**난이도**: ⭐⭐ | **중요도**: ⭐⭐⭐⭐⭐

- 고객 기본 정보 등록
  - 회사명, 대표자, 담당자명, 연락처, 이메일
  - 사업자번호, 주소
  - 메모 기능
- 고객 목록 조회 및 검색
- 고객 상세 정보 조회
- 고객 정보 수정/삭제
- **광고 계정 정보 연결** ⭐ (고객별로 관리)
  - 네이버, 구글, 메타, 카카오 등 매체별 계정 정보
  - 계정 ID, 접근 권한, API 키 (암호화 저장)
  - 매체 계정 만료일 알림
- **보고서 발송 정보** ⭐ (고객별로 관리)
  - 보고서 발송 주기 (주간/월간)
  - 발송 요일/일자
  - 수신 이메일 목록
  - 보고서 템플릿 선택

##### 3. 매장 관리 (기본)
**난이도**: ⭐⭐ | **중요도**: ⭐⭐⭐⭐⭐

- 매장 기본 정보 등록
  - 매장명, 주소, 업종, 전화번호
  - 고객 연결 (1개 고객에 여러 매장 가능)
- 매장 목록 조회 (고객별, 전체)
- 매장 상세 정보 조회
- 매장 정보 수정/삭제

##### 4. 판매 관리 - 상품 마스터
**난이도**: ⭐⭐ | **중요도**: ⭐⭐⭐⭐

- 상품 카테고리 관리
  - 블로그, 카페, 지도 최적화, 키워드 광고 등
- 상품별 기본 단가 설정
- 상품 설명 및 메모

##### 5. 판매 관리 - 견적/주문
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐⭐⭐

- 견적서 생성
  - 고객 선택
  - 매장 선택 (선택사항)
  - 상품 선택 및 수량
  - 고객별 단가 적용 (저장 가능)
  - 합계 자동 계산
  - PDF 다운로드
- 주문 등록
  - 견적서에서 주문 전환
  - 또는 직접 주문 등록
  - 주문 상태 (대기/진행/완료/취소)
- 주문 목록 조회
  - 일/주/월/분기/연 단위 조회
  - 고객별, 매장별, 상품별 필터
  - 엑셀 다운로드

#### 🟡 우선 (2-3개월)

##### 6. 판매 관리 - 계약서
**난이도**: ⭐⭐ | **중요도**: ⭐⭐⭐⭐

- 계약서 파일 업로드
- 계약 정보 등록
  - 고객/매장 연결
  - 계약 기간 (시작일/종료일)
  - 계약 금액
  - 계약 내용 요약
- 계약 만료 알림 (D-30, D-7)

##### 7. 기본 정산 관리
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐⭐⭐

- 세금계산서 발행 기록
  - 주문과 연결
  - 발행일, 공급가, 부가세, 합계
- 입금 여부 수동 체크
  - 입금일, 입금 금액
  - 미수금 자동 계산
- 미수금 현황 조회
  - 고객별 미수금
  - 기간별 미수금
- 월별 매출 요약
  - 총 매출, 입금액, 미수금
  - 고객별, 상품별 매출

##### 8. 보고서 관리 (기본) ⭐
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐⭐

- 보고서 생성
  - 고객/매장 선택
  - 보고 기간 선택
  - 템플릿 선택 (주간/월간)
- 보고서 데이터 수동 입력
  - 성과 데이터 (노출, 클릭, 전환 등)
  - 주요 성과 요약
  - 특이사항 메모
- 보고서 목록 및 이력 관리
  - 발송일, 발송 대상
  - 파일 첨부 (PDF/이미지)
- 보고서 PDF 다운로드

#### 🟢 선택 (3-4개월)

##### 9. 간단한 대시보드
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 이번 달 매출/수익 요약
- 미수금 총액
- 활성 고객/매장 수
- 이번 주 계약 만료 알림

### Phase 1 성공 기준
- ✅ 고객-매장-주문-정산이 하나의 시스템에서 관리됨
- ✅ 견적서/세금계산서 생성 시간 80% 단축
- ✅ 미수금 현황을 실시간으로 파악 가능
- ✅ 고객별 광고 계정 정보를 체계적으로 관리
- ✅ 보고서 발송 이력을 추적 가능

### Phase 1 기술 스택 권장
- **백엔드**: Node.js + Express 또는 Python + FastAPI
- **프론트엔드**: Vue.js 또는 React
- **DB**: PostgreSQL 또는 MySQL
- **인증**: JWT 기반 간단 인증
- **PDF 생성**: jsPDF, pdfmake 또는 reportlab

## 🚀 Phase 2: 효율화 & 자동화 (2-3개월)

### 목표
반복 작업을 자동화하고, 데이터 입력 효율을 높임

### 개발 항목

#### 1. OCR 자동화
**난이도**: ⭐⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 사업자등록증 OCR
  - 사업자번호, 상호, 대표자, 주소 자동 추출
  - 고객 정보 자동 입력
- 명함 OCR
  - 담당자명, 연락처, 이메일 추출
- 영수증 OCR
  - 구매 비용 자동 기록

#### 2. 구매 관리
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 구매 제품 마스터
  - 블로그, 트래픽, 리뷰, 영수증 작업 등
  - 매입 단가
  - 발주 조건
- 거래처 관리
  - 거래처 정보 (업체명, 연락처, 담당자)
  - 거래 내역
- 자동 발주 제안
  - 주문 기반으로 필요 수량 계산
  - 발주서 생성

#### 3. 향상된 정산 기능
**난이도**: ⭐⭐⭐⭐ | **중요도**: ⭐⭐⭐⭐

- 은행 계좌 API 연동
  - 토스뱅크, 오픈뱅킹 등
  - 거래 내역 자동 수집
- 입금 자동 매칭
  - 세금계산서와 입금 내역 자동 연결
  - 미수금 자동 갱신
- 비용 관리
  - 구매 비용 자동 집계
  - 고객별, 매장별 수익률 계산

#### 4. 자동 알림 시스템
**난이도**: ⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 이메일/카카오톡 알림
  - 계약 만료 D-30, D-7
  - 미수금 D+30, D+60
  - 보고서 발송 일정
- 세금계산서 자동 발송
  - 이메일 또는 카카오톡
  - 발송 이력 기록

### Phase 2 성공 기준
- ✅ OCR을 통한 데이터 입력 시간 70% 단축
- ✅ 입금 확인 자동화로 수작업 90% 감소
- ✅ 알림 시스템으로 중요 일정 놓치지 않음
- ✅ 구매-판매 연동으로 수익률 실시간 파악

## 🎨 Phase 3: 고급 기능 & 외부 연동 (2-3개월)

### 목표
데이터 분석 및 외부 시스템 통합으로 경쟁력 강화

### 개발 항목

#### 1. 매장 심화 분석
**난이도**: ⭐⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 크롤러 연동 (place-crawler 프로젝트)
  - 키워드 순위 추적
  - 리뷰 수/평점 변화
  - 경쟁사 분석
- 작업 전/후 비교
  - 순위 변화 그래프
  - 성과 리포트 자동 생성

#### 2. 성과 리포팅 고급 기능 ⭐
**난이도**: ⭐⭐⭐⭐⭐ | **중요도**: ⭐⭐⭐⭐

- 광고 플랫폼 API 연동
  - 네이버 검색광고 API
  - 구글 Ads API
  - 메타 Ads API
- 데이터 자동 수집
  - 노출, 클릭, 전환, 비용 등
  - 일별/주별/월별 자동 집계
- 리포트 자동 생성
  - 템플릿 기반 PDF 자동 생성
  - 그래프/차트 포함
  - 이메일 자동 발송

#### 3. 외부 연동
**난이도**: ⭐⭐⭐⭐⭐ | **중요도**: ⭐⭐

- 전자계약 API (모두싸인 등)
- 세금계산서 API (호두, 더존)
- 구글 캘린더 동기화
- 슬랙/잔디 알림 연동

#### 4. 고급 분석
**난이도**: ⭐⭐⭐⭐⭐ | **중요도**: ⭐⭐⭐

- 고객 생애 가치 (LTV) 분석
- 이탈 위험 고객 예측
- 상품별 수익성 분석
- 시간대별 업무량 분석

### Phase 3 성공 기준
- ✅ 광고 성과 데이터 자동 수집으로 수작업 제로화
- ✅ 보고서 작성 시간 95% 단축
- ✅ 데이터 기반 의사결정으로 수익성 20% 향상

## 📋 Phase 1 제외 항목 (Phase 2-3으로 이연)

다음 기능들은 MVP에 포함하지 않고, 추후 단계에서 구현합니다:

- ❌ 프로젝트/캠페인 진행 관리 → Phase 2-3
- ❌ 작업/일정 통합 캘린더 → Phase 2-3
- ❌ 매체 예산 관리 → Phase 2-3
- ❌ 시간 추적 & 수익성 분석 → Phase 3
- ❌ 리드/영업 관리 → Phase 3
- ❌ OCR 기능 → Phase 2
- ❌ 자동 발주 → Phase 2
- ❌ 크롤러 연동 → Phase 3
- ❌ 광고 API 연동 → Phase 3
- ❌ 외부 API 통합 → Phase 3

**제외 이유**: Phase 1은 핵심 데이터 구조와 기본 워크플로우 구축에 집중합니다. 자동화 및 고급 기능은 기본 시스템이 안정화된 이후 단계적으로 추가합니다.
