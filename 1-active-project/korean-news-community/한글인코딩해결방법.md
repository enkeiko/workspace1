# 한글 인코딩 문제 해결 방법

Windows PowerShell에서 한글이 깨지는 문제를 해결하는 방법입니다.

## 🚀 빠른 해결 방법

### 방법 1: PowerShell 스크립트 사용 (권장)

```powershell
# 개발 서버 실행
.\run-dev.ps1

# 수집 스크립트 실행
.\run-collect.ps1
```

### 방법 2: npm 스크립트 사용

```powershell
# 개발 서버 실행
npm run dev:utf8

# 수집 스크립트 실행
npm run collect:utf8
```

### 방법 3: 수동 인코딩 설정

PowerShell에서 실행하기 전에 다음 명령어를 실행:

```powershell
[Console]::OutputEncoding = [System.Text.Encoding]::UTF8
[Console]::InputEncoding = [System.Text.Encoding]::UTF8
$OutputEncoding = [System.Text.Encoding]::UTF8
chcp 65001 | Out-Null

# 그 다음 실행
npm run dev
# 또는
npm run collect
```

## 🔧 적용된 수정 사항

1. **package.json 스크립트**: UTF-8 인코딩 설정 추가
2. **Node.js 코드**: `process.stdout.setDefaultEncoding('utf8')` 추가
3. **PowerShell 스크립트**: `run-dev.ps1`, `run-collect.ps1` 생성

## ⚠️ 주의사항

- PowerShell에서 직접 `npm run dev`를 실행하면 한글이 깨질 수 있습니다
- 위의 스크립트를 사용하거나 인코딩을 먼저 설정하세요
- VS Code 터미널에서는 기본적으로 UTF-8이 설정되어 있어 문제가 없을 수 있습니다

## 📝 포트 변경

포트 3000이 이미 사용 중이므로 **포트 3002**로 변경되었습니다.

`.env` 파일에서 확인:
```env
PORT=4000
```

서버 접속: http://localhost:4000
