# Improver 재구성

## 입력/출력
- 입력: `clients/{brand}/intake.json`, `collector.json`, `analysis.json`
- 출력: `clients/{brand}/improved.json`

## 기능
- 키워드 제안(간단 결합 규칙 → 후속 강화)
- 소개문 생성(템플릿 기반)
- 4주 뉴스 캘린더(테마/아이디어)
- 리뷰 템플릿 2종(짧은/자세한)
- 비주얼 개선 제안
 - NAP 보정 제안(불일치 항목에 대한 정규화/통일 가이드)

## 원칙
- 검증 불가 수치 사용 금지
- 분석에서 발견된 결측을 개선 제안으로 연결
