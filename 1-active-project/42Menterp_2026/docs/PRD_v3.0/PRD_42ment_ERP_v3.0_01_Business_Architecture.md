## 1. 비즈니스 개요

### 1.1 서비스 설명
네이버 플레이스에 등록된 매장(광고주)들의 리뷰/저장/길찾기 마케팅을 대행하는 ERP 시스템

### 1.2 핵심 비즈니스 플로우
```
고객 계약 → 매장 등록
    ↓
견적서 작성/승인
    ↓
주문/수주 확정
    ↓
상품별 발주 분리 생성
    ↓
상품에 발주서 전송 (Google Sheets)
    ↓
마케팅 작업 진행 (리뷰/저장/길찾기/유입)
    ↓
작업 결과 수집 → 명세 확정
    ↓
정산 (매장 → 본사 → 상품)
    ↓
세금계산서 발행/입금 확인
```

### 1.3 현재 문제점 (AS-IS)
- Excel로 수기 관리 → 누락/오류 빈번
- 발주서 상품별 수동 작성 → 시간 소요
- 정산/계산서 수동 확인 → 누락 위험
- 마케팅 효과 측정 불가 → 키워드 순위 미추적
- 파트너사 관리 어려움 → 정산 복잡

### 1.4 목표 (TO-BE)
- 발주~정산 전 과정 시스템화
- 상품별 발주서 자동 생성 (Google Sheets)
- 세금계산서 자동 발행/수신 (바로빌)
- 키워드 순위 자동 트래킹
- 파트너사 멀티테넌시 지원
