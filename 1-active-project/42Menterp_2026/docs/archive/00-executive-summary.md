## 1. 현재 상황 분석

### 1.1 기존 프로젝트 현황

| 프로젝트 | 기술 스택 | 완성도 | 강점 | 약점 |
|---------|----------|--------|------|------|
| **42ment-erp** | Python + Streamlit + SQLite | 85% | 시간 추적, PDF 청구서, 변경 이력(Adjustment) | 웹 확장 불가, 광고대행 도메인 미지원 |
| **marketing-agency-erp** | Next.js + Prisma + SQLite | 70% | 견적/주문/정산, 광고상품 특성, 상담관리 | 구매관리 없음 = 손익분석 불가 |
| **renewal 기획서** | Next.js + PostgreSQL (계획) | 기획만 | N:M 관계, 보안, 키워드순위 | 미구현, 너무 방대 |

### 1.2 핵심 문제점

```
🔴 Critical: 원가(매입) 관리 없이 광고대행업 ERP는 의미 없음

현재 상태:
- 매출만 있고 비용이 없어서 "얼마 벌었는지" 계산 불가
- 고객에게 100만원 청구하고, 거래처에 70만원 지급하면 → 수익 30만원
- 이 계산이 현재 불가능함
```

### 1.3 각 프로젝트의 활용 가치

| 프로젝트 | 활용 방안 |
|---------|----------|
| 42ment-erp | Adjustment(변경 이력) 시스템 개념 도입 |
| marketing-agency-erp | **메인 코드베이스로 확장** (70% 완성) |
| renewal 기획서 | N:M 관계, 보안 설계 참조 |

## 2. 2026 버전 설계 원칙

### 2.1 비즈니스 우선 (Business First)

ERP는 기술이 아니라 비즈니스 도구입니다.

```
광고대행업 핵심 질문:
1. 이번 달 얼마 벌었는가? (매출)
2. 얼마 나갔는가? (비용)
3. 남은 게 얼마인가? (이익)
4. 받을 돈이 얼마인가? (미수금)
5. 줄 돈이 얼마인가? (미지급금)
```

### 2.2 핵심 업무 플로우

```
상담 → 진단 → 견적 → 계약확정 → 구매발주 → 작업실행 → 검수 → 청구 → 입금 → 손익확정
```

### 2.3 기술 선택 기준

| 항목 | 결정 | 이유 |
|------|------|------|
| 프레임워크 | **Next.js 14+** | 웹 확장성, 이미 70% 구현됨 |
| 데이터베이스 | **SQLite** (MVP) → **PostgreSQL** (Phase 2) | MVP는 SQLite, 멀티유저 시 PostgreSQL |
| ORM | **Prisma** | 타입 안정성, 마이그레이션 |
| UI | **shadcn/ui + Tailwind** | 현재 구현된 것 유지 |
| 언어 | **TypeScript** | 타입 안정성 |

## 3. 2026 버전 핵심 추가 기능

### 3.1 P0 (최우선) - 구매/원가 관리

| 기능 | 설명 | 비즈니스 가치 |
|------|------|--------------|
| Supplier (거래처) | 블로그 업체, 리뷰 업체, 광고 대행사 등록 | 거래처 관리 |
| SupplierProduct | 거래처별 상품 매입 단가 | 원가 파악 |
| PurchaseOrder | 구매 발주서 관리 | 비용 추적 |
| ProfitAnalysis | 주문별 손익 자동 계산 | 경영 의사결정 |

### 3.2 P1 (중요) - 결제/정산 개선

| 기능 | 설명 | 비즈니스 가치 |
|------|------|--------------|
| Payment | 입금 기록 및 계산서 매칭 | 현금흐름 관리 |
| CustomerProductPrice | 고객별 특별 단가 | 가격 정책 |
| Dashboard 개선 | 손익 요약, 미수금 현황 | 경영 현황 파악 |

### 3.3 P2 (권장) - 품질 개선

| 기능 | 설명 | 비즈니스 가치 |
|------|------|--------------|
| AuditLog | 모든 변경 이력 추적 | 데이터 무결성 |
| PDF 템플릿 | 견적서/계산서 PDF | 고객 대응 |
| Excel 내보내기 | 모든 목록 Excel 다운로드 | 데이터 활용 |
