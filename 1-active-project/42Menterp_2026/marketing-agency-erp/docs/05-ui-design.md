## 2. 화면별 기능 설명

### 2.1 메인 대시보드 (`/`)

#### 레이아웃
- **헤더**: 로고, 검색바, 알림, 프로필 (향후)
- **사이드바**: 네비게이션 메뉴
- **메인 콘텐츠**: 대시보드 위젯들

#### 주요 위젯
1. **요약 카드**
   - 총 매출
   - 총 비용
   - 순이익
   - 미수금
   - 선금

2. **매출 차트**
   - 기간별 매출 추이 (선 그래프)
   - 기간 선택 (일/주/월/분기/연)

3. **최근 주문**
   - 최근 10개 주문 목록
   - 주문 번호, 고객명, 금액, 상태
   - 주문 상세로 이동

4. **진행 중인 작업**
   - 우선순위 높은 작업 목록
   - 작업명, 마감일, 상태
   - 작업 상세로 이동

5. **최근 상담**
   - 최근 상담 내역
   - 고객명, 상담 채널, 날짜
   - 상담 상세로 이동

6. **알림**
   - 중요한 알림 목록
   - 미수금 알림, 마감일 알림 등

#### 기능
- 기간 필터 (오늘, 이번 주, 이번 달, 이번 분기, 올해, 사용자 정의)
- 데이터 새로고침
- 위젯 크기 조정 (향후)

### 2.2 고객 목록 페이지 (`/customers`)

#### 레이아웃
- **헤더**: 페이지 제목, 고객 추가 버튼
- **필터 영역**: 검색, 태그 필터, 정렬
- **목록 영역**: 고객 테이블

#### 테이블 컬럼
- 체크박스 (다중 선택)
- 고객명
- 사업자번호
- 담당자
- 연락처
- 태그
- 연결된 매장 수
- 최근 주문일
- 액션 (상세, 수정, 삭제)

#### 기능
- 검색 (이름, 사업자번호)
- 필터 (태그)
- 정렬 (이름, 최근 주문일, 생성일)
- 페이지네이션
- 다중 선택 (일괄 작업)
- Excel 다운로드
- 고객 추가 버튼

### 2.3 고객 상세 페이지 (`/customers/[id]`)

#### 레이아웃
- **헤더**: 고객명, 수정 버튼, 삭제 버튼
- **탭 메뉴**: 정보, 매장, 주문, 상담, 보고서, 문서, 플랫폼 계정

#### 탭별 내용

**1. 정보 탭**
- 기본 정보
  - 고객명
  - 사업자번호
  - 사업자등록증 (파일 다운로드)
  - 담당자
  - 이메일
  - 전화번호
  - 주소
  - 메모
  - 태그
- 통계 정보
  - 총 주문 수
  - 총 매출
  - 평균 주문 금액
  - 최근 주문일

**2. 매장 탭**
- 연결된 매장 목록
- 매장 추가 버튼
- 매장별 통계

**3. 주문 탭**
- 주문 목록 (테이블)
- 주문 추가 버튼
- 주문별 상세 정보

**4. 상담 탭**
- 상담 히스토리 (타임라인)
- 상담 추가 버튼
- 상담별 상세 정보

**5. 보고서 탭**
- 발송된 보고서 목록
- 보고서 생성 버튼

**6. 문서 탭**
- 관련 문서 목록
- 문서 업로드 버튼

**7. 플랫폼 계정 탭**
- 연결된 플랫폼 계정 목록
- 계정 추가 버튼

### 2.4 고객 생성/수정 페이지 (`/customers/new`, `/customers/[id]/edit`)

#### 레이아웃
- **헤더**: 페이지 제목, 저장 버튼, 취소 버튼
- **폼 영역**: 입력 필드들

#### 입력 필드
- 고객명* (필수)
- 사업자번호
- 사업자등록증 파일 업로드
- 담당자
- 이메일
- 전화번호
- 주소
- 메모
- 태그 (다중 선택)

#### 기능
- 실시간 검증
- 저장 시 성공/실패 메시지
- 취소 시 확인 다이얼로그

### 2.5 주문 목록 페이지 (`/orders`)

#### 레이아웃
- **헤더**: 페이지 제목, 주문 추가 버튼
- **필터 영역**: 검색, 고객 필터, 매장 필터, 상태 필터, 기간 필터
- **목록 영역**: 주문 테이블

#### 테이블 컬럼
- 체크박스
- 주문번호
- 고객명
- 매장명
- 주문일
- 납기일
- 총 금액
- 지불 금액
- 미지불 금액
- 상태
- 액션

#### 기능
- 검색
- 필터 (고객, 매장, 상태, 기간)
- 정렬
- 페이지네이션
- Excel 다운로드
- 주문 추가 버튼

### 2.6 주문 상세 페이지 (`/orders/[id]`)

#### 레이아웃
- **헤더**: 주문번호, 상태 배지, 수정 버튼
- **정보 영역**: 기본 정보, 주문 항목, 결제 정보
- **액션 버튼**: 견적서 생성, 세금계산서 생성, 결제 처리

#### 정보 섹션
- 기본 정보
  - 주문번호
  - 고객명 (링크)
  - 매장명 (링크)
  - 주문일
  - 납기일
  - 상태
  - 메모
- 주문 항목
  - 상품명
  - 수량
  - 단가
  - 총액
  - 메모
  - 항목 추가/수정/삭제
- 금액 정보
  - 총 금액
  - 지불 금액
  - 미지불 금액
- 결제 이력
  - 결제일
  - 금액
  - 메모

#### 기능
- 주문 수정
- 상태 변경
- 결제 처리
- 견적서 생성
- 세금계산서 생성
- 주문 삭제

### 2.7 정산 대시보드 페이지 (`/settlements`)

#### 레이아웃
- **헤더**: 페이지 제목, 기간 선택
- **요약 영역**: 요약 카드들
- **차트 영역**: 매출/비용 차트
- **상세 영역**: 상세 테이블

#### 주요 섹션
1. **요약 카드**
   - 총 매출
   - 총 비용
   - 순이익
   - 미수금
   - 선금

2. **기간별 매출/비용 차트**
   - 선 그래프 (매출, 비용, 순이익)
   - 기간 선택 (일/주/월/분기/연)

3. **고객별 매출**
   - 테이블 또는 바 차트
   - 고객명, 매출액, 비율

4. **매장별 매출**
   - 테이블 또는 바 차트
   - 매장명, 매출액, 비율

5. **기간별 상세 내역**
   - 테이블
   - 날짜, 매출, 비용, 순이익

#### 기능
- 기간 선택
- 그룹화 (일/주/월)
- Excel 다운로드
- PDF 다운로드 (향후)

  - 상담 주제
  - 상담 결과 배지
  - 액션 버튼

- **테이블 뷰**: 상담 테이블
  - 상담일시
  - 고객명
  - 채널
  - 주제
  - 결과
  - 액션

#### 기능
- 검색
- 필터
- 정렬
- 뷰 전환 (카드/테이블)
- 상담 추가 버튼

## 5. 반응형 디자인 가이드

### 5.1 브레이크포인트
- **모바일**: < 640px
- **태블릿**: 640px - 1024px
- **데스크톱**: > 1024px

### 5.2 모바일 최적화
- 사이드바: 햄버거 메뉴로 전환
- 테이블: 카드 뷰로 전환
- 필터: 모달로 표시
- 폼: 세로 배치

### 5.3 태블릿 최적화
- 사이드바: 축소된 형태
- 테이블: 가로 스크롤 가능
- 필터: 사이드 패널

### 5.4 데스크톱 최적화
- 전체 레이아웃 표시
- 다중 컬럼 레이아웃
- 호버 효과

## 6. 디자인 시스템

### 6.1 색상 팔레트
- **Primary**: 브랜드 색상
- **Secondary**: 보조 색상
- **Success**: 성공 (녹색)
- **Warning**: 경고 (노란색)
- **Error**: 에러 (빨간색)
- **Info**: 정보 (파란색)
- **Gray**: 회색 스케일

### 6.2 타이포그래피
- **Heading 1**: 페이지 제목
- **Heading 2**: 섹션 제목
- **Heading 3**: 서브 섹션 제목
- **Body**: 본문
- **Caption**: 캡션
- **Label**: 레이블

### 6.3 간격
- 4px 단위 시스템
- 일관된 여백

### 6.4 그림자
- 카드 그림자
- 모달 그림자
- 드롭다운 그림자

### 6.5 애니메이션
- 페이드 인/아웃
- 슬라이드
- 호버 효과
- 로딩 스피너

## 7. 접근성 고려사항

### 7.1 키보드 네비게이션
- 모든 인터랙티브 요소에 키보드 접근 가능
- Tab 순서 논리적
- Enter/Space로 액션 실행

### 7.2 스크린 리더
- 적절한 ARIA 레이블
- 의미 있는 HTML 시맨틱 태그
- 이미지에 alt 텍스트

### 7.3 색상 대비
- WCAG AA 기준 준수
- 색상만으로 정보 전달하지 않음

### 7.4 포커스 표시
- 명확한 포커스 인디케이터
- 키보드 포커스 시각적 표시
