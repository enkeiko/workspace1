# 42ment ERP - Windows 실행 패키지

## 빠른 시작 (3단계)

1. **압축 해제**: ZIP 파일을 원하는 위치에 압축 해제
2. **실행**: `시작-42ment-ERP.bat` 파일을 더블클릭
3. **ERP 시작**: GUI 런처에서 "▶ ERP 시작" 버튼 클릭

## 이 패키지에 포함된 것

```
42ment-ERP-Windows/
├── app/                          # 애플리케이션 파일
│   ├── 42ment-ERP.exe           # GUI 런처 실행 파일
│   ├── src/                      # ERP 소스 코드
│   ├── config/                   # 설정 파일
│   ├── data/                     # 데이터 폴더
│   │   ├── 42ment.db            # 데이터베이스 (최초 실행 시 생성)
│   │   ├── exports/             # 내보내기 파일
│   │   ├── logs/                # 로그 파일
│   │   └── backups/             # 백업 파일
│   └── _internal/               # 의존성 파일 (수정 금지)
│
├── 시작-42ment-ERP.bat          # 간편 실행 스크립트
├── README-설치.txt               # 설치 및 사용 가이드
├── QUICKSTART.md                # 빠른 시작 가이드
└── GUI_사용법.md                # GUI 사용 방법
```

## 시스템 요구사항

- **운영체제**: Windows 10/11 (64-bit)
- **메모리**: 최소 4GB RAM
- **디스크**: 500MB 여유 공간
- **기타**:
  - 인터넷 연결 불필요 (완전 오프라인 동작)
  - Python 설치 불필요 (모든 것이 포함됨)

## 설치 방법

### 1. 압축 해제

- ZIP 파일을 마우스 오른쪽 버튼 클릭
- "압축 풀기" 또는 "Extract All..." 선택
- 원하는 위치 선택 (예: `C:\42ment-ERP`)

⚠️ **주의**:
- 한글이 포함된 경로도 가능하지만, 영문 경로 권장
- OneDrive, Dropbox 등 클라우드 동기화 폴더는 피하세요

### 2. 첫 실행

1. `시작-42ment-ERP.bat` 파일을 더블클릭
2. GUI 런처가 열립니다
3. "데이터베이스 초기화" 버튼 클릭 (최초 1회)
4. (선택) "샘플 데이터 로드" 버튼 클릭 (데모용)
5. "▶ ERP 시작" 버튼 클릭

### 3. 웹 브라우저 접속

- 자동으로 브라우저가 열립니다
- 또는 수동으로: http://localhost:8501

## 사용 방법

### GUI 런처 사용

**ERP 시작**
- "▶ ERP 시작" 버튼 클릭
- 브라우저가 자동으로 열립니다
- 시스템이 실행 중일 때는 버튼이 비활성화됩니다

**ERP 중지**
- "⬛ 중지" 버튼 클릭
- 또는 런처 창을 닫으면 자동 종료

**데이터베이스 도구**
- **초기화**: 데이터베이스를 새로 만듭니다 (기존 데이터 삭제)
- **샘플 데이터**: 예제 데이터를 로드합니다 (테스트용)

### 직접 실행 (고급)

GUI 런처 없이 직접 실행하려면:

```bash
cd app
42ment-ERP.exe
```

   - 예: `42ment_backup_20250114.db`

## 문제 해결

### Windows Defender 경고

**문제**: "Windows에서 PC를 보호했습니다" 메시지

**해결**:
1. "추가 정보" 클릭
2. "실행" 버튼 클릭
3. (선택) 폴더를 "제외 항목"에 추가

### 앱이 시작되지 않음

**문제**: 더블클릭해도 아무 일도 일어나지 않음

**해결**:
1. 바이러스 백신 프로그램 확인
2. 관리자 권한으로 실행
3. 이벤트 뷰어에서 오류 로그 확인

### 브라우저가 열리지 않음

**문제**: 런처에서 시작했지만 브라우저가 안 열림

**해결**:
1. 수동으로 브라우저 열기
2. 주소창에 입력: `http://localhost:8501`
3. 포트가 사용 중인지 확인: `netstat -ano | findstr 8501`

### 포트 충돌

**문제**: "Address already in use" 오류

**해결**:
1. 다른 프로그램이 8501 포트 사용 중
2. 해당 프로그램 종료
3. 또는 포트 변경 (고급 사용자)

### 데이터가 사라짐

**문제**: 이전 데이터를 찾을 수 없음

**해결**:
1. `app\data\backups` 폴더 확인
2. 다른 PC나 폴더에 설치했는지 확인
3. 백업 파일이 있다면 복원

## 업데이트

### 새 버전 설치

1. 기존 데이터 백업:
   - `app\data\42ment.db` 파일 복사
   - `app\data\exports` 폴더 복사

2. 새 버전 압축 해제 (다른 폴더에)

3. 데이터 복원:
   - 백업한 파일을 새 버전의 `app\data`에 복사

### 버전 확인

- GUI 런처 하단에 버전 표시
- 또는 `app\README.md` 파일 확인

## 제거 (언인스톨)

1. GUI 런처에서 "⬛ 중지" 클릭
2. 런처 종료
3. 설치 폴더 전체 삭제

⚠️ **주의**: 데이터도 함께 삭제됩니다. 필요한 경우 먼저 백업하세요.

## 고급 설정

### 포트 변경

`app\src\main.py` 파일 수정 (고급 사용자만)

## 자주 묻는 질문 (FAQ)

**Q: 인터넷 연결이 필요한가요?**
A: 아니요. 완전 오프라인으로 동작합니다.

**Q: 여러 PC에서 사용할 수 있나요?**
A: 네. 데이터베이스 파일을 복사하면 됩니다.

**Q: 동시에 여러 사용자가 사용할 수 있나요?**
A: 아니요. 단일 사용자용입니다. 동시 접근 시 데이터 충돌 가능.

**Q: Mac이나 Linux에서도 되나요?**
A: 이 패키지는 Windows 전용입니다. 다른 OS는 소스 코드로 설치하세요.

**Q: 데이터는 안전한가요?**
A: 모든 데이터가 로컬에 저장됩니다. 외부 전송 없음.

**Q: 업데이트는 자동인가요?**
A: 아니요. 수동으로 새 버전을 다운로드해야 합니다.
