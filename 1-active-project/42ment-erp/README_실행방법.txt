🚀 가장 쉬운 방법 (권장)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. START_ERP.bat 파일을 더블클릭하세요

  2. GUI 창이 열립니다

  3. "▶ ERP 시작" 버튼을 클릭하세요

  4. 자동으로 브라우저가 열립니다!

  끝! 정말 쉽죠? 😊

  실행 파일 종류
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✅ START_ERP.bat     → 배치 파일 (콘솔 창 보임)
  ✅ START_ERP.vbs     → 무음 실행 (콘솔 창 없음)
  ✅ launcher.py       → Python GUI 런처


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  처음 사용 시 (최초 1회)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. START_ERP.bat 더블클릭

  2. "패키지 설치하시겠습니까?" → "예" 클릭
     (Streamlit, Pandas 등 자동 설치됨)

  3. 설치 완료 후 GUI에서:
     - [데이터베이스 초기화] 버튼 클릭
     - [샘플 데이터 로드] 버튼 클릭

  4. [▶ ERP 시작] 버튼 클릭

  5. 브라우저가 자동으로 열립니다!


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  GUI 런처 기능
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ▶ ERP 시작            → ERP 시스템 시작
  ⬛ 중지               → ERP 시스템 중지
  [데이터베이스 초기화]  → DB 새로 만들기 (⚠️ 데이터 삭제됨)
  [샘플 데이터 로드]     → 테스트용 샘플 데이터 추가


━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  브라우저 주소
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  자동으로 안 열리면 브라우저에서 직접 입력하세요:

  http://localhost:8501

  문제 해결
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ❌ Python 오류
     → Python 3.11+ 설치 필요
     → https://www.python.org

  ❌ 패키지 설치 실패
     → 명령 프롬프트 열고:
       pip install streamlit pandas

  ❌ 포트 충돌
     → GUI에서 [⬛ 중지] 버튼 클릭
     → 다시 [▶ ERP 시작] 클릭
