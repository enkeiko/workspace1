## 🛠️ 기술 스택

### Backend
- Python 3.11+
- SQLite (로컬 DB)
- Pandas (데이터 처리)

### Frontend
- Streamlit 1.28+
- Tkinter (GUI 런처)

### Architecture
- 3-Layer: UI → Service → Model → Database
- RESTful 패턴
- Repository 패턴
