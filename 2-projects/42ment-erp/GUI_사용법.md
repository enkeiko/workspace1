# 42ment ERP - GUI 실행 가이드

## 🚀 빠른 시작

### 방법 1: GUI 런처 사용 (권장) ⭐

**가장 쉬운 방법!**

1. `START_ERP.bat` 파일을 더블클릭
2. GUI 창이 열리면 "▶ ERP 시작" 버튼 클릭
3. 자동으로 브라우저가 열립니다!

### 방법 2: 무음 실행 (백그라운드)

1. `START_ERP.vbs` 파일을 더블클릭
2. 콘솔 창 없이 GUI가 바로 열립니다
3. "▶ ERP 시작" 버튼 클릭

### 방법 3: Python으로 직접 실행

```bash
python launcher.py
```

### 2️⃣ 주요 버튼 설명

#### ▶ ERP 시작
- ERP 시스템을 시작합니다
- 자동으로 브라우저가 열립니다 (http://localhost:8501)
- 시작 후에는 버튼이 비활성화됩니다

#### ⬛ 중지
- 실행 중인 ERP를 중지합니다
- 중지 확인 메시지가 표시됩니다
- 안전하게 서버를 종료합니다

#### 데이터베이스 초기화
- 데이터베이스를 처음부터 다시 만듭니다
- ⚠️ **주의**: 기존 데이터가 모두 삭제됩니다!
- 백업을 먼저 하세요

#### 샘플 데이터 로드
- 테스트용 샘플 데이터를 추가합니다
- 고객 3명, 프로젝트 2개 등이 추가됩니다
- 데이터베이스가 비어있을 때 유용합니다

## ❗ 문제 해결

### 문제 1: "Python is not installed" 오류

**원인**: Python이 설치되지 않았거나 PATH에 없음

**해결**:
1. Python 3.11+ 설치: https://www.python.org
2. 설치 시 "Add Python to PATH" 체크
3. 컴퓨터 재시작

### 문제 2: 패키지 설치 실패

**원인**: 인터넷 연결 문제 또는 pip 문제

**해결**:
```bash
# 명령 프롬프트에서 직접 실행
pip install streamlit pandas
```

### 문제 3: 브라우저가 자동으로 열리지 않음

**원인**: 브라우저 설정 문제

**해결**:
- 브라우저를 직접 열고
- `http://localhost:8501` 접속

### 문제 4: "포트가 이미 사용 중" 오류

**원인**: ERP가 이미 실행 중이거나 다른 프로그램이 포트 사용

**해결**:
1. [⬛ 중지] 버튼 클릭
2. 작업 관리자에서 python.exe 프로세스 종료
3. 다시 [▶ ERP 시작] 클릭

### 문제 5: GUI 창이 안 열림

**원인**: tkinter 누락 (드물게 발생)

**해결**:
- Python 재설치 (tkinter 포함 옵션 선택)
- 또는 명령 프롬프트에서 직접 실행:
  ```bash
  streamlit run src/main.py
  ```

## 📱 바로가기 만들기

### Windows 데스크탑 바로가기

1. `START_ERP.bat` 파일에 마우스 우클릭
2. "바로가기 만들기" 선택
3. 바로가기를 데스크탑으로 이동
4. 바로가기 이름을 "42ment ERP"로 변경

### 아이콘 변경 (선택사항)

1. 바로가기에 마우스 우클릭
2. "속성" 선택
3. "아이콘 변경" 클릭
4. 원하는 아이콘 선택

## 🎯 시작 시 자동 실행 설정

ERP를 컴퓨터 시작 시 자동으로 실행하려면:

1. `Win + R` 키 입력
2. `shell:startup` 입력 후 Enter
3. 시작 프로그램 폴더가 열림
4. `START_ERP.vbs` 파일의 바로가기를 이 폴더에 복사
